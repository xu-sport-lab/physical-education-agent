### 2.3 全动作库进退阶系统

**新增函数**：
- `getExerciseLevel(ex, lastTest)` - 根据体测得分返回难度等级
  - < 50分 → regression（退阶）
  - 50-79分 → base（标准）
  - ≥ 80分 → progression（进阶）
- `getExerciseAtLevel(ex, level)` - 获取指定难度下的动作参数
- `applyLevels(exArr)` - 批量应用难度调整到动作数组

**覆盖范围**：全部 90 个动作均已添加 `levels` 字段，包含 regression 和 progression 变体

### 2.5 进度追踪系统

**存储方式**：localStorage（前缀：`cycle_progress_{studentId}`）

**功能**：
- 每次训练后勾选"标记为已完成"
- 可填写训练备注
- 自动保存日期和训练状态
- 支持加载已保存的进度

**新增函数**：
- `toggleSessionProgress(sessionKey, weekNum, dayName)` - 切换训练完成状态
- `loadCycleProgress(studentId)` - 加载已保存的进度

### 2.6 第4周末体测对比与自动调整

**自动调整逻辑**：
1. 各项成绩提升 ≥ 10分 → 下周期该素质降至维持量
2. 各项成绩提升 < 5分或下降 → 下周期加大该素质训练比重并降低难度
3. 总分提升 ≥ 15分 → 下周期整体难度等级上调一级
4. 总分提升 < 5分 → 下周期重新评估训练目标和动作选择策略

**数据存储**：
- 基线体测：`cycle_baseline_{studentId}`
- 调整建议：`cycle_advice_{studentId}`

**新增函数**：
- `compareCycleResults()` - 对比体测结果并生成下周期建议
- `generateCycleOverview()` - 生成周期总览数据

### 2.7 PDF 导出功能

**新增函数**：
- `exportTrainingPlanPDF()` - 导出单次训练方案为 PDF
- `exportFourWeekPlanPDF()` - 导出4周周期计划为 PDF
- `buildPlanPDFHTML(innerContent, student, isFourWeek)` - 构建 PDF 专用 HTML（含排版美化样式）

**PDF 排版特性**：
- 渐变色页头（紫色系）
- 章节式布局
- 动作卡片式展示
- 时间线可视化
- 颜色编码的难度/标签
- 页脚水印

**按钮更新**：

- "📄 导出PDF"（红色按钮）
